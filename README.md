# JCModa — Loja de Moda Online

Projecto académico desenvolvido no âmbito da disciplina de Desenvolvimento Web,
na Universidade. Este site representa uma loja de roupas e calçados fictícia
baseada em Luanda, Angola.

## Sobre o Projecto

O site JCModa foi criado com o objectivo de aplicar os conhecimentos adquiridos
em sala de aula, cumprindo os seguintes critérios de avaliação:

- Conteúdo adequado ao público-alvo
- Definição de cores e posicionamento
- Escrita persuasiva
- Boa experiência do utilizador
- Foco no cliente
- Identidade visual da empresa
- Integração com redes sociais
- Imagens e gráficos de qualidade
- Mapa do site
- Navegação simples e rápida
- Optimização para motores de busca (SEO)
- Responsividade para todos os dispositivos
- Desempenho e tempo de carregamento

## Tecnologias Utilizadas

- HTML5
- CSS3 puro (sem frameworks)
- JavaScript 

## Estrutura do Projecto
jcmoda/
├── css/          → Folhas de estilo (uma por página)
├── html/         → Páginas HTML
└── img/          → Imagens em formato SVG e png

## Páginas do Site

- Home, Sobre Nós, Produtos, Galeria

- Termos, Privacidade, Mapa do Site, 404
